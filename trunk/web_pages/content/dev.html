Developer Links

Mailing lists

We have two mailing lists for OpenRecord developers. One is a development discussion list, where people can ask questions and make suggestions. The other list just has automatically generated messages that are sent by the Subversion repository when anyone makes changes to the source code. Anyone can subscribe to either list. You can also just browse through the archives of past messages.

  • developer mailing list: openrecord-dev
  • subversion mailing list: openrecord-svn

Source code: downloading

You should be able to download the OpenRecord source code using any Subversion client. You can download the source code via anonymous SVN, so you don't need to have any sort of account.

  • You can download a Subversion client from: subversion.tigris.org
  • For info about using Subversion to get the OpenRecord source, see the OpenRecord SVN page
  • Using a Subversion client, the OpenRecord source code is available at: "svn://svn.berlios.de/openrecord"

Source code: contributing

The OpenRecord source code repository is being hosted by BerliOS. In order to commit changes to the source code, you will need to have a developer account at BerliOS, and you will need me (Brian Skinner) to add you to the list of developers associated with the OpenRecord project.

  • general information about BerliOS
  • register for a BerliOS developer account
  • Once you have a developer account and are listed as an OpenRecord developer, the source code is available at: "svn+ssh://svn.berlios.de/svnroot/repos/openrecord"
